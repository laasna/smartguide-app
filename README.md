A React-based multilingual chat application that helps users find the perfect gadget through conversational AI.

## Features

- **Multilingual Support**: English, Hindi, Telugu, Spanish, German, French, Marathi
- **Smart Recommendations**: Personalized gadget suggestions based on usage, budget, and preferences
- **Modern UI**: Glassmorphism design with blurred professional background
- **Conversational Flow**: Step-by-step guided questions for better recommendations
- **Mobile Responsive**: Works seamlessly on desktop and mobile devices

## How It Works

1. **Language Selection**: Choose your preferred language
2. **Gadget Type**: Specify what you're looking for (phone, laptop, tablet, etc.)
3. **Usage Context**: Tell us how you'll use it (work, study, gaming, travel)
4. **Requirements**: Share specific needs or let AI suggest ideal specs
5. **Budget**: Set your price range
6. **Brand Preference**: Choose a brand or get the best overall recommendation
7. **Get Recommendation**: Receive one perfect product match with buy link

## Tech Stack

- React 18
- CSS3 with Glassmorphism effects
- Google Fonts (Inter)
- Responsive design

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm start
   ```

3. Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

## Build for Production

```bash
npm run build
```
